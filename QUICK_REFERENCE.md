# 📋 Quick Reference Guide

A quick reference for common tasks and metrics in the Retail Store Profit Analysis project.

---

## 🎯 Key Metrics at a Glance

| Metric | Description | Formula |
|--------|-------------|---------|
| **Total Revenue** | Sum of all sales income | ∑(Units Sold × Unit Price) |
| **Total Cost** | Sum of all costs | ∑(Units Sold × Unit Cost) |
| **Total Profit** | Net profit | Total Revenue - Total Cost |
| **Profit Margin** | Profitability percentage | (Total Profit / Total Revenue) × 100 |
| **Average Order Value** | Average revenue per order | Total Revenue / Total Orders |

---

## 📊 Dashboard Components

### KPI Cards
- **Total Orders:** Count of all transactions
- **Total Revenue:** Sum of all revenue
- **Total Cost:** Sum of all costs
- **Total Profit:** Sum of all profits

### Charts

| Chart Name | Type | Purpose |
|------------|------|---------|
| Top 10 Profit by Country | Horizontal Bar | Identify best markets |
| Profit by Region | Pie/Donut | Regional comparison |
| Profit by Item Type | Bar Chart | Product performance |
| Profit by Sales Channel | Pie Chart | Online vs Offline |
| Yearly Profit Trend | Line Chart | Time series analysis |
| Order Priority | Various | Priority distribution |

## 🔍 Common Analysis Tasks

### Task 1: Find Top Performing Products
1. Look at "Profit by Item Type" chart
2. Or use Item Type slicer to compare individual categories
3. Check both revenue and profit metrics

### Task 2: Identify Best Markets
1. Review "Top 10 Profit by Country" chart
2. Look at "Profit by Region" for broader view
3. Consider expanding operations in top regions

### Task 3: Compare Sales Channels
1. Check "Profit by Sales Channel" chart
2. Note the percentage split
3. Consider resource allocation based on performance

### Task 4: Analyze Time Trends
1. Examine "Yearly Profit Trend" chart
2. Look for growth patterns
3. Identify seasonal variations

## 🎛️ Slicer Guide

### Using Item Type Slicer
- **Single Selection:** Click once
- **Multiple Selection:** Ctrl + Click
- **Clear Filter:** Click filter icon (funnel)
- **Select All:** Ctrl + A

### Effects of Filtering
When you filter by Item Type:
- ✅ All charts update automatically
- ✅ KPI cards recalculate
- ✅ Pivot tables refresh
- ✅ Dashboard shows filtered view

## 🔧 Quick Fixes

### Chart Not Updating?
```
Right-click chart → Refresh
Or: Data → Refresh All
```

### Slicer Not Working?
```
Right-click slicer → Report Connections
Check pivot table connections
```

### Slow Performance?
```
Formulas → Calculation Options → Manual
Press F9 to calculate when needed
```

### Need to Add Data?
```
1. Update CSV file
2. Data → Refresh All
3. Check pivot tables
```
